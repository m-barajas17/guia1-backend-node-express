# Backend Node.js y Express

Proyecto de práctica para el taller de Git y control de versiones.

## Descripción
API backend construida con Node.js y Express como parte del programa SENA ADSO.

## Cómo ejecutar
\`\`\`
npm install
node src/app.js
\`\`\`
